# VideoParse - 视频解析附属

## 支持解析类型

- [x] Bilibili

## 指令

`/parse <type> <url>`

- `<type>` 为连接类型，可选字段为：`bili`
- `<url>` 为视频链接，暂仅支持解析以下类型
  - bv号链接 
    - PC端: `https://www.bilibili.com/video/BV1bRiMeREPE`
    - 移动端: `https://m.bilibili.com/video/BV15b42177rL`
  - 手机端分享链接: `https://b23.tv/RxFlKKf`
