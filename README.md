# InfiniteBot Expansion v4

> 注意：在使用附属前，请务必浏览附属对应的配置文件(config.yml)

第四代 `InfiniteBot` 插件官方附属拓展，基于 ib4 框架，附属天然支持

- 热部署/热卸载
- 配置文件/语言文件版本化
- 跨端通信 (基于 [perpetua-sdk](https://github.com/IUnlimit/perpetua-sdk-for-java))
